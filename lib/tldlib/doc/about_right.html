Top Level Domain (TLD) library : About

tldlib is a library for looking up TLDs (top level domains). It provides hashes that assign descriptions to

  • all TLDs used on the Internet
  • country-code TLDs
  • non-country TLDs (separate hashes for old and new ones also exist)
  • for RFC 2606 (BCP 32) reserved TLDs.

It also provides functions that check whether a TLD falls in any of the categories listed before.

tldlib was written by Josef 'Jupp' Schugt, jupp@sourceforge.net.
