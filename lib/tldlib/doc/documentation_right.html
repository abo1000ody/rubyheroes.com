Top Level Domain (TLD) library : Documentation

Constants

tldlib defines a number of hashes that assign meanings to TLDs that are actively used on the Internet as well as to those special-purpose TLDs that are defined in Request For Comment 2606 (RFC 2606) which is also known as Best Current Practice 32 (BCP 32).

TLD::All
Assigns descriptions to all valid TLDs.
Does not include TLD::RFC_2606 (TLD::BCP_32).
Created by joining TLD::Country and TLD::Noncountry.
Corresponds to TLD::valid?.
TLD::Country
Assigns meanings to all TLDs that are assigned to countries.
Corresponds to TLD::country?.
TLD::Noncountry
Assigns meanings to all TLDs not assigned to a country.
Created by joining TLD::Noncountry_old and TLD::Noncountry_new
Corresponds to TLD::noncountry?.
TLD::Noncountry_old
Assigns meanings to arpa, com, edu, gov, int, mil, net, and org.
Corresponds to TLD::noncountry_old?.
TLD::Noncountry_new
Assigns meanings to aero, biz, coop, info, museum, name, and pro.
See also TLD::noncountry_new?.
TLD::RFC_2606
Assigns meanings to example, invalid, localhost and test
These TLDs are reserved for local use, testing purposes and documentation (see RFC 2606 for details).
Identical to TLD::BCP_32
Corresponds to TLD::rfc_2606?.
TLD::BCP_32
Identical to TLD::RFC_2606.
Corresponds to TLD::bcp_32?.

Methods

The methods to find out whether a TLD is valid, is a country-code TLD, etc. are implemented by simply checking whether the TLD is member of the hash in question.

TLD::valid?
Shorthand for TLD::All.member?
TLD::country?
Shorthand for TLD::Country.member?
TLD::noncountry?
Shorthand for TLD::Noncountry.member?
TLD::noncountry_old?
Shorthand for TLD::Noncountry_old.member?
TLD::noncountry_new?
Shorthand for TLD::Noncountry_new.member?
TLD::rfc_2606?
Shorthand for TLD::RFC_2606.member?
TLD::bcp_32?
Same as TLD::rfc_2606?
Shorthand for TLD::BCP_32.member?
